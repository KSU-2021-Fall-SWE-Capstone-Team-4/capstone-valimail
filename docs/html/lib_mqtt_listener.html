MQTT Listener

class lib.mqtt_listener.MQTTListener
begin_listening()

Loops forever (starts the listening loop). Required call for actually listening to stuff. Acts as a wrapper for paho.mqtt.client.Client.loop_forever. This is where the __main__ process ends, since loop_forever should never break UNLESS the timeout was achieved.

static on_message(client, user_data, msg)

Callback method for receiving a message through self.client. Used as a static method here so that self.client can use it.

Parameters

  • client (paho.mqtt.client.Client) – The client calling this method.

  • user_data – The user data for the established connection.

  • msg (paho.mqtt.client.MQTTMessage) – The message object.

static on_subscribe(client, user_data, mid, granted_qos)

Callback method for subscription through self.client. Used as a static method here so that self.client can use it.

Parameters

  • client (paho.mqtt.client.Client) – The client calling this method.

  • user_data – The user data for the established connection.

  • mid – Currently unknown.

  • granted_qos (bool) – Whether or not the desired quality of service level has been granted.

subscribe(topic, qos=0, options=None, properties=None)

Subscribes to a topic. Acts as a wrapper for paho.mqtt.client.Client.subscribe.

Parameters

  • topic (str) – The topic name.

  • qos (int) – Desired quality of service level. Defaults to 0.

  • options – Currently unknown.
